# CKD Prediction (Flask)

## Run (Windows / PowerShell)

This repo includes a virtual environment in `.venv` with the required dependencies (including `scikit-learn`).

Option A (no activation needed):

```powershell
.\.venv\Scripts\python.exe app.py
```

Option B (activate venv):

```powershell
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python app.py
```

If PowerShell blocks activation scripts, you can still run with Option A.
